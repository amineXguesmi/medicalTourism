# UI Tokens

Shared design tokens for web and mobile.

Planned contents:

- colors,
- spacing,
- typography scale,
- radius,
- semantic status colors,
- accessibility constants.

Keep tokens domain-appropriate: calm, clinical, readable and built for repeated operational use.

The first MVP token set is implemented in `src/index.ts`. Mobile and web should consume the same semantic roles rather than inventing feature-local colors.
